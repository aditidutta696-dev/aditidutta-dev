## üöÄ FEATURED PROJECT DEEP-DIVES

### 1. Cross-Platform Threat Mitigator (AI-SOAR)
**The Problem:** Traditional EDR (Endpoint Detection and Response) can be heavy or limited by OS silos.
**The Solution:** An intelligent bridge between Windows and Kali Linux.
* **The Brain:** Uses an **Isolation Forest** unsupervised learning model to analyze network packet flow and system metrics. It detects anomalies without needing a database of "known signatures."
* **The Action:** Upon detection of an anomaly (e.g., unauthorized port binding), the Windows agent triggers a secure SSH signal to a Kali Linux "Mitigator" instance.
* **Outcome:** Automated IP blacklisting and service isolation in under **2 seconds** from detection.

### 2. Spring4Shell (CVE-2022-22965) Research & Exploitation
**Objective:** Deconstructing one of the most significant Java vulnerabilities of the decade.
* **Methodology:** Built a containerized lab using **Docker** to simulate a vulnerable Spring Framework environment.
* **Exploitation:** Crafted a Python payload to manipulate class loaders and achieve Remote Code Execution (RCE).
* **Defense:** Developed a mitigation script that patches the vulnerability at the application layer and logs attempt patterns to a SIEM.

### 3. Security-Monitoring-Agent (Python)
**Focus:** Host-based Intrusion Detection (HIDS).
* **Monitoring:** Tracks file integrity (MD5/SHA256 hashing) of system files in real-time.
* **Logging:** Exports JSON-formatted logs ready for ingestion by ELK Stack or Splunk.
* **Stealth:** Designed to run as a low-footprint daemon, making it difficult for basic malware to detect its presence.
